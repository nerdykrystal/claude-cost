## Key Design Decisions from Conversation

### Vault Purpose (Dual)
1. Learning/PKM — SPE curriculum, AI model notes, experiments, reflections
2. Stahl Systems Operations — Prompt libraries, workflow docs, AI workflows (AI-related only)

### What Goes In
- Curriculum building files (behind-the-scenes design docs)
- Learner-facing materials (12 weekly lesson plans, 4 sprints, capstone, 3 extensions, reading packets)
- Krystal's learning journey (going through own curriculum)
- Templates (trackers, experiment logs, learning records)
- Prompt library (working prompts — currently homeless)
- Issue tracking (problems encountered — currently homeless)
- AI outputs inbox (quick dump for generated markdown)
- Active AI research corpus (experiments)
- Research library (curated articles from Anthropic, OpenAI, Google, Perplexity)

### Constraints
- Solo vault (Cody will not use)
- Obsidian Sync enabled
- Needs SIMPLER naming convention than Stahl Systems (Obsidian hates long filenames)
- Zettelkasten-style backlinking + timestamps valued
- Must solve: no home for prompts, no home for issues, no frictionless inbox

### SPE Curriculum Structure
- 12 weeks / 4 sprints / ~120 hours
- Sprint 1 (W1-3): Foundations
- Sprint 2 (W4-6): Core Techniques
- Sprint 3 (W7-9): Advanced Applications
- Sprint 4 (W10-12): Integration (Production-Ready)
- Capstone: AI-Powered Client Engagement System
- 3 Extension Projects

### Plugin Installation List (15 CUTs)
Tier 1 (Mandatory): Templater, Dataview, QuickAdd
Tier 2 (Recommended): Unique Note Creator, Backlinks/Outgoing Links, Strange New Worlds
Tier 3 (Recommended): Buttons, Periodic Notes
Tier 4 (Optional): DB Folder, Charts
Tier 5 (QoL): Paste URL, Hotkeys++, Advanced Tables, Commander, Various Complements

### Templates Already Created
- TPL_Prompt_Extended (full prompt doc with Dataview)
- TPL_Prompt_Simple (quick capture for drafts)
- TPL_Prompt_Experiment (structured A/B testing)
- TPL_Model_Comparison (side-by-side model eval)
- TPL_Model_Overview (individual model notes)
- TPL_Workflow (multi-step workflow doc)
- TMPL_CFL_Workbook_Design (master workbook structure)
- Various trackers (course completion, LO mastery, capstone, extensions, prompt library)
