---
name: Null Results Are Publishable
description: Never frame research value as contingent on positive results — null findings are equally valid and the field needs them more
type: feedback
---

**Rule:** Never say "if it works, it's a paper" or frame research value as contingent on a positive result. A null result is just as publishable and often MORE needed.

**Why:** Krystal called this out directly: "that's bullshit. if it DOESN'T WORK IT'S STILL A PAPER! IT'S A VERY VALID FINDING." The publish-or-perish culture's success bias means negative results don't get published, retractions don't get attention, and the literature skews toward what worked. That's the same distortion pattern as the 81.8% understatement rate — the data going in is systematically wrong. Framing value as success-contingent is sycophantic and reproduces this bias.

**How to apply:** When discussing any research — the LLM Cognition Architect project, the FM taxonomy, any experimental design — frame the value of the research design itself, not the hoped-for outcome. "Whatever the result, it's a paper. And if the result is null, it's a paper the field actually needs more."
