---
name: Funnel Design Rules
description: Krystal corrected landing-page thinking — portfolios use true multi-page funnels, one CTA per page, not single-page designs
type: feedback
---

A landing page is NOT a funnel. A funnel moves through MULTIPLE pages with only one CTA exit per page. Each page of a website must have a singular purpose.

**Why:** In the Portfolio Designer session, Claude designed a single-page layout with all 5 pain points. Krystal corrected: a hiring manager is hiring for ONE pain point, not all 5. Dense single-page designs overload visitors and dilute the funnel's conversion power.

**How to apply:** When designing any portfolio or marketing site for Krystal, never propose a single-page design with multiple CTAs. Route visitors through a sequence of pages, each with one purpose and one exit action. The PORT1 funnel uses 2D routing: Problem Routing (rank pain points) x Depth Routing (3 time tracks).
