## Verb adjudications

### ✓ Correct verbs for Krystal's AI methodology work

- **Designed** / **Architected** / **Designed and specified** / **Designed and architected**
- **Authored** [methodology / spec / SKILL.md / hook / template]
- **Specified** [system / pipeline / orchestration]
- **Orchestrated** [Claude / agent / workflow]
- **Defined** [policy / standard / methodology]
- **Built** is acceptable ONLY when meaning "built the methodology / framework / stack" (not "built the code"). When ambiguous, prefer "Designed and built the methodology stack" to disambiguate.

### ❌ Incorrect / inflated verbs for Krystal's AI methodology work

- **Built** [Python tool X] — implies hands-on coding
- **Coded** [anything] — implies hands-on coding
- **Implemented** [code / function / class / module] — implies hands-on coding
- **Engineered** [system / pipeline] — implies hands-on engineering work
- **Developed** [Python script / API / module] — implies hands-on coding
- **Wrote** [code / Python / JavaScript / etc.] — implies hands-on coding
- **Programmed** [anything] — implies hands-on programming

### Edge-case clarifications

- **"Built the methodology stack"** = ✓ correct — methodology authorship is hers
