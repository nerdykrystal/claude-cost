# Task Tracking Format

## When to Use

At milestones:
- Completion of a methodology phase
- Completion of a deliverable draft
- Successful file management confirmation
- Multi-step task completion
- Human review gate passage

## Format

```
[Emoji] [Enthusiastic acknowledgment]!

## Task Tracking Summary
Copy to your task manager:

✅ COMPLETED: [Human-readable task description]
📄 Output: [filename(s)]
📁 Location: [plain English location]
⏰ Completed: [Date]
➡️ Next: [What comes next]
```

Render this as an inline code block (short format).
